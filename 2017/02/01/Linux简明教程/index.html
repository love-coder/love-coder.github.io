Linux 简明教程

发表于 2017-02-01 | 分类于 操作系统

最近工作中一些几乎没有Linux基础的同学在开发过程中遇到一些问题，所以我花了两天时间整理了这些基础知识供其查找，以满足日常工作需要。文章分为四个部分：Linux目录结构、常用软件安装方法、vim基本使用，并在最后整理了30条左右的常用指令。因为定位是简明教程，所以不会涉及过多原理以及对每条指令展开阐述。该文章均是以64位CentOS 7.1为例，其他发行版也基本类似。

Linux目录结构

Linux系统中所有资源均是以文件的方式被表示。常见的有：普通文件、目录文件、字符设备文件和块设备文件、符号链接文件等。一般认为Linux是没有扩展名的概念，也就是说一个文件能不能被执行跟其扩张名没有一点关系，和在Windows平台下的.exe、.bat等不同。只要用户账户具备x即执行权限，就可以执行该文件，当然能不能执行成功就得看具体内容。但是，为了能够更好地理解Linux，开源社区任然建议遵循约定俗成的一些规范，比如.sh是脚本或批量处理文件，.tar.gz、.zip、.tar等是压缩文件，.html是网页文件等等。同样，被约定俗成的还有它的目录结构，它的各部分含义如下：

常用指令

下面列出的均是常用的指令，在熟悉一段时间后，一方面是能记得这些指令，更重要的是熟练使用help指令自查。

cd – Change Directory

格式：cd [目录名]

功能：切换当前工作目录，比如进入根目录cd /，进入当前工作目录的上一级目录cd ..。

举例：pwd指令为列出当前具体工作目录

ls – List

格式：ls[选项] [目录名]

功能：列出目标目录的子文件或目录

常用参数：

-a或-all 列出所有文件，包括.开头的隐藏文件
-r或-reverse 反向列举
-R或-recursive 同事递归列出所有子目录
-l 列出文件的详细详细(ls - l 等同于ll )
